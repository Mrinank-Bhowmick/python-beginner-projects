## Internet Speed Tester
Check the Download and Upload Speed of your Internet Connection.

### Requirements.
1. pyspeedtest
2. speedtest
3. speedtest-cli

## Example

```text
{'url': 'http://speedtest.lon1.linode.com:8080/speedtest/upload.php', 'lat': '51.5085',
 'lon': '-0.1257', 'name': 'London', 'country': 'United Kingdom', ...}
94371840.25
8921034.11
```

The three lines show the best server details, the download speed in bits/second, and the upload speed in bits/second.

### How to use
```
./main.py
```
