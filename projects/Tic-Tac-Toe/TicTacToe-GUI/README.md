# Tic-Tac-Toe (GUI)

A graphical Tic-Tac-Toe game built with `pygame`. `tic tac.py` provides a mouse-driven board with X/O image tiles and a computer opponent; `oneplayermode.py` is a terminal-driven variant of the same game logic.

## How to run

```bash
pip install pygame
python "tic tac.py"
```

## Dependencies

- pygame

## Pyodide-runnable

No — it depends on `pygame`, which opens a desktop window and cannot run in a browser.
