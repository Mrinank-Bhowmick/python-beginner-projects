# Snake Game

A classic Snake game built with Pygame. The player controls a snake that grows as it eats food, tracking a score and a high score, with game-over and play-again screens.

## How to run

```
pip install pygame
python src/main.py
```

## Dependencies

pygame

## Pyodide-runnable

No — it uses Pygame for windowed graphics and real-time keyboard input, which require a desktop display.
