# Space Shooter

A 2D space shooter game built with Pygame. Pilot a spaceship, fire lasers at incoming enemy ships, dodge enemy lasers and homing missiles, and rack up a score, with menu and game-over screens.

## How to run

```
pip install pygame
python main.py
```

## Dependencies

pygame

## Pyodide-runnable

No — it uses Pygame for windowed graphics and loads image/font assets, which require a desktop display.
